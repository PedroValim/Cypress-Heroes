BUG-001 – Login inválido não exibe mensagem clara
--------------------------------------------------

Título:
Login inválido não exibe mensagem de erro clara ao usuário

Descrição:
Ao tentar realizar login com credenciais inválidas, o sistema não exibe
uma mensagem clara informando o motivo da falha, o que pode gerar confusão
para o usuário.

Pré-condições:
- Aplicação em execução
- Usuário na tela de login

Passos para reprodução:
1. Acessar a aplicação Cypress Heroes
2. Informar usuário inválido
3. Informar senha inválida
4. Clicar no botão "Login"

Resultado esperado:
O sistema deve exibir uma mensagem clara informando que as credenciais
são inválidas.

Resultado obtido:
O sistema não exibe mensagem clara ou exibe mensagem genérica.

Severidade:
Média

Prioridade:
Média

Status:
Aberto

BUG-002 – Usuário sem permissão visualiza ações administrativas
--------------------------------------------------

Título:
Usuário sem permissão consegue visualizar botões administrativos

Descrição:
Usuários sem perfil administrativo conseguem visualizar botões de
criação, edição ou exclusão de heróis, o que fere as regras de permissão
da aplicação.

Pré-condições:
- Usuário comum autenticado

Passos para reprodução:
1. Realizar login com usuário sem permissão administrativa
2. Acessar a listagem de heróis
3. Abrir o detalhe de um herói

Resultado esperado:
Usuários sem permissão NÃO devem visualizar botões administrativos
(criar, editar ou excluir herói).

Resultado obtido:
Botões administrativos são exibidos para usuários comuns.

Severidade:
Alta

Prioridade:
Alta

Status:
Aberto
