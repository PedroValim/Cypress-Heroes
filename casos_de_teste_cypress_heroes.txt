CT-001 – Login com usuário válido
-----------------------------------------------------
Funcionalidade:
Autenticação de Usuário

Objetivo:
Verificar se um usuário válido consegue realizar login com sucesso no sistema.

Pré-condições:
- Aplicação Cypress Heroes em execução
- Usuário válido cadastrado
- Navegador aberto na tela de login

Dados de Teste:
Usuário: admin
Senha: admin123

Passos:
1. Acessar a aplicação Cypress Heroes
2. Informar o usuário válido no campo Username
3. Informar a senha válida no campo Password
4. Clicar no botão Login

Resultado Esperado:
- O sistema deve autenticar o usuário
- O usuário deve ser redirecionado para a tela de listagem de heróis

Tipo de Teste:
Funcional / Automatizado

Status:
(A definir após execução)

CT-006 – Controle de permissão (UI)
-----------------------------------------------------
Funcionalidade:
Controle de Permissões

Objetivo:
Verificar se usuários sem permissão não visualizam ações restritas na interface.

Pré-condições:
- Usuário autenticado com perfil restrito

Passos:
1. Realizar login com usuário sem permissão
2. Acessar a tela de heróis

Resultado Esperado:
- Botões de criar, editar ou excluir herói não devem ser exibidos
- O sistema deve respeitar o nível de permissão do usuário

Funcional
